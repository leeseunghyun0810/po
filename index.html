ABOUT

WHO IS LEE SH?

안녕하세요, 소통하는 웹퍼블리셔를 꿈꾸는 이승현입니다. 2022년 8월 웹퍼블리셔라는 직업을 알게 되어 그 후 웹퍼블리셔가 되기 위한 지식과 기술을 향상시키기 위해 끊임없이 노력하고 있습니다. 더 많은 배움을 얻고자 학원을 다니며 기초 지식을 쌓았습니다. 웹 접근성과 웹 표준을 준수하는 웹퍼블리셔가 뒤기 위해 성실히 HTML, CSS, JavaScript 등 여러 가지 기술들을 익히고 배워나가고 있습니다.

언제나 성실하고 열심히 배울 준비가 되어 있습니다. 초심 잃지 않고 열심히 하는 웹 퍼블리셔가 되겠습니다!

교육 및 작업 내용

Photoshop, Illustrator, Figma 등 디자인 도구의 기본적인 활용이 가능합니다. Illustrator를 이용해서 직접 로고를 제작하고 Figma를 활용하여 모바일 페이지 UI/UX를 디자인하여 웹 페이지 제작을 하였습니다.

웹의 기초적인 개념과 웹 페이지 작성의 기본이 되는 HTML을 이해하고 기본 태그와 속성, 시맨틱 태그를 잘 사용해서 사용자들이 원활하게 웹 페이지를 사용할 수 있도록 웹 접근성에 기반한 마크업 구조를 작성합니다. 또한 웹 표준을 준수해서 다양한 브라우저와 환경에서 웹 페이지가 정상 작동할 수 있도록 코드를 작성하기 위해 노력하고 있습니다.

CSS3를 활용하여 웹 페이지의 스타일링과 레이아웃을 꾸미는 법을 배우며 CSS3를 통해 사용자에게 더욱더 다양한 경험과 시각적 만족을 제공합니다. 다양한 스타일 속성, 애니메이션, 반응형 등 다양한 기능을 웹 페이지에 적용할 수 있습니다. CSS3의 전처리기인 Sass를 활용해 작업 시간 단축 및 코드 관리를 보다 효율적으로 할 수 있습니다.

JavaScript, JQuery, vue.js, Bootstrap 등 다양한 라이브러리와 프레임워크를 통해 더욱 편리하고 다양한 웹 기능을 사용할 수 있습니다.
